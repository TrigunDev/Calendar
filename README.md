# 📅 Interactive Wall Calendar 

An interactive, visually rich **React + Next.js** calendar component inspired by a physical wall calendar.  
This project focuses on **UI/UX design, responsiveness and interactivity** mimicking a real-world calendar while adding modern features.

---

## 🚀 Live Demo
[Click here](https://calendar-rosy-theta.vercel.app/)

---

## ✨ Features

### 📌 Core Features
- 🗓️ Wall Calendar Aesthetic  
  Designed to resemble a physical wall calendar with a hero image and integrated month/year display.

- 📅 Date Range Selection  
  Select start and end dates with visual highlighting.  

- 📝 Notes Section  
  Add notes to selected dates. 

- 📱 Fully Responsive Design  
  - Desktop: Side-by-side layout  
  - Mobile: Stacked layout  

---

### 🎨 Creative Enhancements
- Smooth animations and transitions. 
- Interactive date highlighting.
- Light and Dark mode.
- Clean UI with modern design.  

---

## 🛠️ Tech Stack

- Frontend: React / Next.js  
- Styling: Tailwind CSS  
- State Management: React Hooks

## ⚙️ Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/TrigunDev/Calendar
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```
